Projects

TrustWave

A chat Fraud Detection System

My project is a dynamic fraud detection web application developed using Python and Flask framework. The application aims to identify potentially fraudulent messages submitted by users. By leveraging the power of Trustwave's fraud detection algorithm, the application analyzes the content of messages and assigns a severity level based on the likelihood of fraud. The project utilizes SQLAlchemy for database management and incorporates user-friendly interfaces using HTML templates and CSS styling.

RECLINE

AI Based Chatbot for mentally disturbed People

Recline is an AI-based chatbot developed to support individuals facing mental health challenges. With a user-friendly web interface and an emotionally intelligent approach, Recline serves as a virtual friend, offering comfort and relief to those suffering from depression and anxiety. Developed using Python and JavaScript, with libraries such as Flask, NLTK, and Torch, the project delivers a working model hosted on a local server. With Recline, I demonstrate my expertise in AI, web development, and using technology to make a positive impact on mental well-being.
